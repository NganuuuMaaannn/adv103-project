# Employment Matching System

# Installation

Clone Repository in CMD
```
https://github.com/NganuuuMaaannn/adv103-project.git
cd adv103-project
code .
```

Create venv
```
python -m venv venv
```

Activate venv
```
venv/Scripts/activate
```

Install Requirements
```
pip install -r requirements.txt
```

Run system
```
python manage.py runserver
```
